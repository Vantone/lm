# 物料批量导入说明

## 📁 导入模板文件

系统支持两种格式的批量导入：
- **CSV 格式**：`批量导入模板.csv`
- **JSON 格式**：`批量导入模板.json`

## 🔄 导入步骤

1. **下载模板**：从项目根目录复制对应的模板文件
2. **编辑数据**：按照模板格式添加您的物料数据
3. **导入系统**：
   - 登录系统
   - 进入"实时仓库管理"页面
   - 点击右上角的 **上传图标 (FileUp)**
   - 选择编辑好的文件进行导入

## 📝 CSV 格式要求

**表头格式（支持中英文）：**
```csv
名称,单位,规格,分类,标签
```

**字段说明：**
- `名称/Name`：物料名称（必填）
- `单位/Unit`：计量单位（必填）
- `规格/Spec`：规格说明（必填）
- `分类/Category`：物料分类（可选，默认"其他"）
- `标签/Tags`：物料标签（可选）

**示例数据：**
```csv
名称,单位,规格,分类,标签
特级面粉,kg,25kg/袋,食材,重要原料
洗发水,瓶,500ml,日用品,洗漱品
```

## 📄 JSON 格式要求

**完整数据结构：**
```json
{
  "materials": [
    {
      "id": "unique-id",
      "name": "物料名称",
      "unit": "单位",
      "spec": "规格",
      "category": "分类",
      "tags": "标签"
    }
  ]
}
```

**或者直接是物料数组：**
```json
[
  {
    "id": "unique-id",
    "name": "物料名称", 
    "unit": "单位",
    "spec": "规格",
    "category": "分类",
    "tags": "标签"
  }
]
```

## ⚠️ 注意事项

1. **ID 字段**：系统会自动生成唯一ID，模板中的ID仅作参考
2. **数据验证**：导入前请确保必填字段完整
3. **重复检查**：系统会根据ID避免重复导入
4. **编码格式**：CSV 文件请使用 UTF-8 编码
5. **备份建议**：导入前建议先导出当前数据作为备份

## 🔧 故障排除

**导入失败常见原因：**
- 文件格式不正确（非 CSV 或 JSON）
- 缺少必填字段（名称、单位、规格）
- CSV 文件编码不是 UTF-8
- JSON 语法错误

**解决方法：**
- 使用提供的模板文件
- 检查必填字段是否完整
- 用文本编辑器验证 JSON 语法
- 确保文件编码为 UTF-8
