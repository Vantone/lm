# Excel 列式导入说明

## 📋 什么是列式导入？

列式导入是指将数据按列组织，每列数据竖向排列：

### 📊 列式格式示例

**Excel 文件布局：**
```
A列          B列          C列          D列
序号          物料名称      标签          基本计量单位
1             特级面粉      重要原料      kg
2             花生油        液态调味品    L
3             白砂糖        甜味基础      kg
```

而不是传统的表格格式：
```
序号 | 物料名称 | 标签     | 基本计量单位
1    | 特级面粉 | 重要原料 | kg
2    | 花生油   | 液态调味品 | L
3    | 白砂糖   | 甜味基础 | kg
```

## 📁 模板文件

- **`Excel导入模板-列式.csv`** - 列式CSV格式模板

## 🎯 列式导入优势

1. **便于复制粘贴**
   - 可以直接从其他系统复制列数据
   - 粘贴到Excel对应列中

2. **数据录入方便**
   - 专注于单一类型数据录入
   - 减少横向移动，提高效率

3. **支持部分导入**
   - 可以只更新某一列数据
   - 系统智能识别并合并

## 📋 数据列说明

| 列位置 | 字段名     | 说明               | 必填 |
|----------|------------|--------------------|--------|
| A列      | 序号       | 记录序号           | 可选  |
| B列      | 物料名称   | 物料名称           | 必填  |
| C列      | 标签         | 物料标签           | 可选  |
| D列      | 基本计量单位 | 基本计量单位       | 必填  |
| E列      | 物料单位   | 规格/包装单位      | 可选  |
| F列      | 昨日库存   | 昨日库存数量       | 可选  |
| G列      | 今日入库   | 今日入库数量       | 可选  |
| H列      | 车间出库   | 车间出库数量       | 可选  |
| I列      | 店面出库   | 店面出库数量       | 可选  |
| J列      | 今日库存   | 今日库存数量       | 可选  |
| K列      | 日期         | 记录日期           | 可选  |

## 🚀 导入步骤

1. **准备Excel文件**
   - 按列格式排列数据
   - 确保每列数据完整

2. **上传文件**
   - 选择 `.xlsx` 或 `.xls` 文件
   - 系统自动识别列式格式

3. **自动处理**
   - 系统智能转换为表格数据
   - 自动合并相同物料的信息

4. **验证结果**
   - 检查导入的物料数量
   - 确认库存数据正确

## ⚠️ 注意事项

1. **数据对齐**
   - 确保每列数据对应正确的物料
   - 序号不是必须，但有助于数据对应

2. **空值处理**
   - 空单元格会被忽略
   - 必填字段不能为空

3. **格式要求**
   - Excel 2007+ (.xlsx) 或 97-2003 (.xls)
   - 建议使用 UTF-8 编码

系统会自动检测您的Excel格式（表格式或列式），并正确处理导入！
